# Student Registration System

## Overview
A simple responsive student registration system built with HTML, CSS and vanilla JavaScript.
Features:
- Add, edit, delete student records.
- Input validation (name letters only, ID numbers only, valid email, contact ≥ 10 digits).
- Persistent storage via `localStorage`.
- Dynamic vertical scrollbar for display table.
- Responsive: mobile / tablet / desktop.

## Project structure
- `index.html` — main page
- `styles.css` — styling
- `script.js` — JavaScript functionality
- `README.md` — this file

## GitHub & commit instructions (assignment requirement)
- Make separate commits for:
  - `index.html`
  - `styles.css`
  - `script.js`
  - `README.md`
- Push to a public GitHub repo and include the link with your submission.

## How to run
1. Put all files in a single folder.
2. Open `index.html` in a browser.
3. Add students, refresh to confirm data persists.
